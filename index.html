Hulixerian Game Engine Editor

This project aimed for making scene creation easier and faster for HGE applications.
HGE is a game engine that use OpenGL for rendering and written by C++11 and it is completely cross-platform.

Project aims

0- Using new pipeline of OpenGL.
1- Using most update techniques and technologies.
2- Using Linux and more specifically Gnome environment.
3- Keeping code as simple as possible.
4- Keeping good design for making future changes easier.
